When Are Homicides Most Common?

Author

Tommy Gavagan

Introduction

Homicide is often discussed using yearly totals, but those numbers hide patterns within the year. Instead of asking only where violence happens, this project asks a simpler question: when does it happen most often?

Why this matters

If homicides consistently peak in certain months, that suggests the pattern is not random. Understanding timing can help show more broad behavioral/environmental influences.

Dataset overview

This dataset includes reported homicide cases across multiple U.S. cities, each with a reported date. By grouping cases by month, we can compare how counts change across the calendar year. (https://www.kaggle.com/datasets/joebeachcapital/homicides/data)

The question

Do homicides follow a seasonal pattern? And does that pattern change by city?

Explore the data

Use the dropdown to switch between all cities and individual cities. Hover to see exact counts.

What stands out

The distribution is not uniform. Some months clearly have higher counts than others.

  • Highest month: July
  • Lowest month: February

Why the city filter matters

The overall trend might hide differences across cities. Some cities may have sharper peaks, while others remain more consistent throughout the year.

Use the dropdown to test this.

Open exploration

This graphic raises a couple follow-up questions:

  • Are warmer months consistently higher?
  • Do certain cities drive the pattern?
  • Would results change if adjusted for population?

The goal is not to answer everything, but to let you explore patterns freely.

Conclusion

By focusing on monthly patterns instead of yearly totals, this visualization reveals a clearer and more interactive view of when homicides occur. The addition of city-level filtering allows users to move beyond averages and explore variation across locations. ```
